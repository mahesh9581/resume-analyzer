# 🤖 ResumeAI — Smart Resume Analyzer v4.0

Full-stack AI Resume Analyzer using React + Node.js + Express

---
url:https://cozy-lollipop-7bb66c.netlify.app/

## 🚀 Setup & Run

### Step 1 — Backend
```bash
cd backend
npm install
cp .env.example .env
# Open .env and add your API key
npm run dev
```

### Step 2 — Frontend
```bash
cd frontend
npm install
npm start
```

Open 👉 http://localhost:3000

---

## .env Setup
```
AI_API_KEY=your_api_key_here
PORT=5000
```

## ✨ Features
- Upload PDF, DOCX, TXT resume
- Paste resume text directly
- Job description matching
- 4 animated score rings
- Corrections with severity levels
- ATS keyword analysis
- Skills gap analysis
- Action verb detector
- Quantification scoring
- Top 3 priority fixes
